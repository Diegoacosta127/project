What is Holberton School?

What's the purpose of the project?

As applicants, students must create a website from scratch. First they have to configure the server where the site'll be allocated, and then they have to write the website using HTML, CSS and JavaScript.

What's next?

Students will have an intensive learning in programming for 9 months. Ther they'll see programming languages as C, Python and SQL for databases.
